  ## **安装方式**  
  
Laravel 需要 PHP &gt;= 7.1以上的版本，并且同时需要PHP安装以下扩展

```
- cURL extension

- mbstring

- BC Math
```
使用 ` git ` 将代码clone到本地，导入最新的数据库文件(迁移文件等会)，然后运行

```
composer update
```
在`.env`中配置好数据库和redis等。

顺便生成secret,`php artisan key:generate`和`php artisan jwt:secret`,
